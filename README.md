# 고해영 일병 구하기

savinghaey.co.kr 개인 업무 보조 정적 웹사이트입니다.

## 포함 기능

- `/work-scheduler/` — 업무 스케쥴러
- `/links/` — 업무 사이트 바로가기 허브
- `/memo/` — 메모 작성 및 A4 출력/PDF 저장용 상세페이지

## 저장 방식

- 기본 소스 데이터는 저장소 파일에서 관리합니다.
- 사용자가 화면에서 추가한 일정, 링크, 메모는 서버가 아니라 브라우저 localStorage에 저장됩니다.
- PC 변경 또는 브라우저 초기화 전에는 각 기능의 JSON 백업 기능을 사용하세요.
